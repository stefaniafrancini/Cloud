# Mi Ropero

**Mi Ropero** es una aplicación web full stack que permite organizar y crear prendas, crear outfits y recibir recomendaciones mediante un agente de inteligencia artificia.  
El proyecto utiliza **Django** para el backend, **SQL Server en Azure** para la Base de datos, **Angular** para el frontend y está completamente desplegado en la nube con **Render** y **Vercel**.

---

## Tecnologías Utilizadas

### **Backend**
- Django
- Autenticación con SimpleJWT  
- Base de datos: **SQL Server alojado en Azure**  
- Despliegue del backend: **Render**

### **Frontend**
- Angular
- Consumo de API REST  
- Despliegue del frontend: **Vercel**

### **Inteligencia Artificial**
- Se detalla en el repositorio del agente
---

## Funcionalidades

### **Gestión de Prendas**
- Crear Prendas
- Subir imágenes  
- Clasificación por categorías  
- Editar y eliminar prendas  

### **Outfits**
- Crear outfits combinando múltiples prendas  
- Visualizar y organizar outfits 

### **Chat con IA**
- Sugerencias inteligentes basadas en los outfits 
- Búsqueda por categoria

### **Usuarios**
- Registro, login y autenticación  
- Cada usuario gestiona su propio ropero
